Details

This program, given the pANinIyan code for a pratyAhAra, fetches the corresponding letters. This is produced from an attempt to convert pANinIya sUtras into a modern programming language (Scala).

Limitations:
1. Encoding of pANini's saMjJNAs is at a very immature state.
2. There are a few errors in naming svaras.
